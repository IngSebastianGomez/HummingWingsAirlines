Manual de instalación Backend HummingWings Airlines

Requisitos

  • Sistema operativo (Windows, Mac, Linux)
  • Python
  • Gestor de paquetes de Python PIP
  • PostgreSQL
  • Navegador web

Clonar repositorio

Clonar el siguiente repositorio de GitHub en una terminal con el comando.

Git clone https://github.com/IngSebastianGomez/HummingWingsAirlines.git

Crear entorno virtual de python e instalar requirements

Crear entorno virtual

python -m venv nombreEntornoVirtual

Activar entorno virtual

Windows

nombreEntornoVirtual\Scripts\activate

Mac/Linux

source nombreEntornoVirtual/bin/activate

Instalar requirements.txt

En una terminal dirigirse a la carpeta Backend donde estará el archivo llamado “requirements.txt” una vez estemos aquí ejecutaremos el siguiente comando

pip install -r requirements.txt

Abrir la carpeta en un editor de código

En la ruta Backend/HummingWings/HummingWings/ en el archivo settings.py cambiar el usuario y contraseña de acceso a la base de datos postgres, tenga en cuenta que estos datos son personales

DATABASES  = {
'default': {
'ENGINE': 'django.db.backends.postgresql_psycopg2',
'NAME': 'local_humming_wings',
'USER': 'Usuario', #Su usuario
'PASSWORD': 'Password', # Su contraseña
'HOST': 'localhost',
'PORT': '5432',
	}
}

Crear base de datos local

Crear en postgres la base de datos local con el nombre que se encuentra en el atributo ‘NAME’ del archivo de configuración en este caso “local_humming_wings”

Realizar migraciones

En una terminal dentro de la carpeta donde está el archivo manage.py ejecutar el comando

python manage.py makemigrations

Migrar

En una terminal dentro de la carpeta donde está el archivo manage.py ejecutar el comando

python manage.py migrate

Crear super usuario de Django

En una terminal dentro de la carpeta donde está el archivo manage.py ejecutar el comando y seguir instrucciones de creación de súper usuario

python manage.py createsuperuser

Ejecutar servidor de django

En una terminal dentro de la carpeta donde está el archivo manage.py ejecutar el comando

python manage.py runserver

Ingresar al modo de administrador y crear variables de entorno

En un navegador ingrese la siguiente dirección en la barra de busqueda

http://localhost:8000/admin/

Se mostrará la siguiente vista donde ingresará los datos del súper usuario creado previamente
